Instructions for uploading texts

For each Kanikau, download the page pdf and rename it to match the following format YYYYMMDD[Newspapertitle].PDF Placing years first ensures that the texts will be sorted chronologically.

To create the text file, first copy the full text from papakillo. From the Kanikau uploads page, select "Create New file," paste the contents into the github page, and name the file YYYYMMDDNewspaper.txt
